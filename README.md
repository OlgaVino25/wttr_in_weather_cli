# Погодный консольный клиент

Простой скрипт на Python для получения прогноза погоды через сервис [wttr.in](https://wttr.in/). Позволяет быстро узнать погоду в любых городах, достаточно изменить список в коде.

**Функциональность**

- Получение погоды для заданных городов (по умолчанию Лондон, аэропорт Шереметьево, Череповец)
- Вывод информации на русском языке
- Компактный формат без лишних управляющих последовательностей
- Лёгкое расширение списка городов — просто добавьте название в массив

**Технологии**

- Python 3.x
- requests — выполнение HTTP-запросов к API [wttr.in](https://wttr.in/)

### Установка и запуск

1. Клонируйте репозиторий.

2. Установите зависимости:

```powershell
pip install requests
```

3. Настройка списка городов

По умолчанию скрипт выводит погоду для Лондона, аэропорта Шереметьево (код svo) и Череповца.
Вы можете легко изменить или дополнить список в коде:

```python
cities = ['London', 'svo', 'Череповец']  # замените на свои варианты
```

Поддерживаются названия городов на любом языке, а также коды аэропортов (например, `svo`, `led`) и другие локации, понимаемые сервисом [wttr.in](https://wttr.in/).

4. Запустите скрипт

```powershell
python main.py
```
